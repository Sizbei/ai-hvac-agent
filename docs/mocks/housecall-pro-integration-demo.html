Setup Requirements

⚠️ MAX Plan Required
Housecall Pro MAX plan is required for API access. The public REST API is not available on lower tiers. Verify your account plan before proceeding.
🔑 API Key Generation
Generate API key in HCP:
  1. Log into Housecall Pro
  2. Go to Settings > API Access
  3. Create a new API key
  4. Copy the key (you'll only see it once!)
🔗 Connect Integration
Connect via Admin Panel:
  1. Navigate to Admin > Chatbot Settings
  2. Find "Housecall Pro" section
  3. Paste your API key
  4. Click "Connect Housecall Pro"
🌐 Register Webhook
Set up webhook for status updates:
https://ai-hvac-agent-lovat.vercel.app/api/webhooks/housecall

Register this URL in your Housecall Pro webhook settings to receive real-time job status updates.

Environment Variables (Optional)

You can also set the API key via environment variable for the default organization:

HOUSECALL_API_KEY=your_api_key_here

Note: The admin panel method is preferred as it's organization-specific and doesn't require redeployment.
