```typescript
}
```

Dialog content:
- Title: "Add Technician" (create) or "Edit Technician" (edit)
- Fields:
  - Name: Input (required)
  - Email: Input type="email" (required)
  - Password: Input type="password" (required in create mode, hidden in edit mode — don't allow password changes here)
  - Active: Switch component (only in edit mode)
- Each field has a Label component
- "Save" Button with loading state
- Form validation: check required fields before submit

Submit handler:
- Create mode: POST `/api/admin/technicians` with `{ name, email, password }`
- Edit mode: PATCH `/api/admin/technicians/${technician.id}` with `{ name, email, isActive }`
- On success: call `onSuccess()` and `onClose()`
- On error: show error message inline

Use useState for form fields. Pre-populate from `technician` prop when in edit mode (useEffect on technician change).
