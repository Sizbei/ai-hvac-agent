**src/hooks/use-admin-requests.ts:**
Custom hook for fetching and polling request list:
```typescript
'use client';

import { useState, useEffect, useCallback, useRef } from 'react';
import type { AdminRequest } from '@/lib/admin/types';

interface UseAdminRequestsOptions {
  readonly status?: string;
  readonly page?: number;
  readonly limit?: number;
}

interface UseAdminRequestsResult {
  readonly requests: readonly AdminRequest[];
  readonly total: number;
  readonly isLoading: boolean;
  readonly error: string | null;
  readonly refetch: () => Promise<void>;
}
```

Implementation:
1. `useState` for requests, total, isLoading, error
2. `fetchRequests` callback that builds URL with query params, calls `fetch('/api/admin/requests?...')`, parses response
3. `useEffect` that calls fetchRequests on mount and when options change
4. `useEffect` that sets up polling interval every 10 seconds: `setInterval(fetchRequests, 10_000)`. Clean up on unmount. Use `useRef` for interval ID.
5. Do NOT poll while a request is in-flight (use a `useRef<boolean>` flag for `isFetching`)
6. Export the hook
