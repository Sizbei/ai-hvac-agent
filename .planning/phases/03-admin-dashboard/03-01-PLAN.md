**src/lib/auth/config.ts:**
Use `jose` library (already compatible with Edge Runtime) to sign and verify JWTs.
- `getAuthSecret()` — reads AUTH_SECRET env var, throws if missing or < 32 chars
- `signToken(payload: AdminSessionPayload): Promise<string>` — creates JWT with HS256, 24h expiry, includes all payload fields in claims
- `verifyToken(token: string): Promise<AdminSessionPayload | null>` — verifies JWT, returns payload or null on failure (do NOT throw on invalid token)

Use `new SignJWT(payload).setProtectedHeader({ alg: 'HS256' }).setIssuedAt().setExpirationTime('24h').sign(encodedKey)` pattern.
Use `jwtVerify(token, encodedKey, { algorithms: ['HS256'] })` for verification.

The `jose` library is NOT in package.json yet. Add it: `npm install jose`.

**src/lib/auth/session.ts:**

**src/app/api/auth/login/route.ts:**
POST handler that:
1. Parses JSON body with Zod: `z.object({ email: z.string().email(), password: z.string().min(1) })`
2. Queries users table: `db.select().from(users).where(eq(users.email, validated.email))` — get first result
3. If no user found, return `errorResponse('Invalid credentials', 'INVALID_CREDENTIALS', 401)`
4. If user.role !== 'admin', return `errorResponse('Access denied', 'FORBIDDEN', 403)`
5. If !user.isActive, return `errorResponse('Account disabled', 'ACCOUNT_DISABLED', 403)`
6. Compare password with bcrypt: `bcrypt.compare(validated.password, user.passwordHash)`
7. If mismatch, return `errorResponse('Invalid credentials', 'INVALID_CREDENTIALS', 401)`
8. Call `createAdminSession({ userId: user.id, organizationId: user.organizationId, email: user.email, name: user.name, role: 'admin' })`
9. Return `successResponse({ user: { id: user.id, name: user.name, email: user.email } })`

Use `import bcrypt from 'bcryptjs'` (already in dependencies).

Do NOT use withTenant here — login validates user across the system, then the session carries organizationId for subsequent requests.

**src/app/api/auth/logout/route.ts:**
POST handler that:
1. Calls `deleteAdminSession()`
2. Returns `successResponse({ message: 'Logged out' })`
