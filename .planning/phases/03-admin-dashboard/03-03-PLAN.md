4. **Submit handler**:
   ```typescript
   async function handleSubmit(e: React.FormEvent) {
     e.preventDefault();
     setIsLoading(true);
     setError('');
     try {
       const res = await fetch('/api/auth/login', {
         method: 'POST',
         headers: { 'Content-Type': 'application/json' },
         body: JSON.stringify({ email, password }),
       });
       const data = await res.json();
       if (!res.ok) {
         setError(data.error?.message ?? 'Login failed');
         return;
       }
       window.location.href = '/admin/requests';
     } catch {
       setError('Network error. Please try again.');
     } finally {
       setIsLoading(false);
     }
   }
   ```
   Use `window.location.href` (not router.push) to ensure the proxy picks up the new cookie on navigation.

5. **Accessibility**: Labels on inputs, form has `onSubmit`, button has `disabled={isLoading}`, autoFocus on email input.

Use shadcn Card, CardHeader, CardTitle, CardDescription, CardContent components for the login card. Use shadcn Input and Button. Use Label from the newly installed label component.
